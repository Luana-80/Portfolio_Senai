# Portfólio – Luana Lucera

Este repositório apresenta meu portfólio pessoal desenvolvido como atividade do curso de Desenvolvimento de Software do SENAI.

O objetivo deste projeto é apresentar meu perfil profissional, minhas habilidades e alguns projetos desenvolvidos durante meu processo de aprendizagem na área de tecnologia.

---

## Sobre mim

Sou professora de Artes por formação e atuei por vários anos na área da educação, experiência que contribuiu para o desenvolvimento de habilidades como organização, criatividade, comunicação e resolução de problemas.

Atualmente estou em transição de carreira para a área de tecnologia e venho me dedicando aos estudos em desenvolvimento de software.

Tenho interesse em:

- Desenvolvimento Web
- Criação de aplicações
- Soluções digitais
- Programação front-end e back-end

Este portfólio reúne alguns projetos e experiências desenvolvidos ao longo do meu processo de aprendizagem.

---
